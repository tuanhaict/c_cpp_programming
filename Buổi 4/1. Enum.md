# Enum

Trước đây các kiểu dữ liệu là có sẵn, bây giờ ta có thể định nghĩa nó thông qua `enum`

Giả sử ta đang làm việc với ngày trong tuần, cần có các giá trị để kiểm tra 1 ngày là ngày nào. Ta có thể định nghĩa các hằng số tương ứng với các ngày trong tuần.
```c
const int MONDAY = 0;
const int TUESDAY = 1;
const int WEDNESDAY = 2;
const int THURSDAY = 3;
const int FRIDAY = 4;
const int SATURDAY = 5;
const int SUNDAY = 6;
```

Có cách đơn giản hơn là dùng `enum`:
```c
enum WeekDay {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

int main()
{
    enum WeekDay day = Monday;
    if (day == Monday)
    {
        printf("Hôm nay là thứ 2");
    }
    return 0;
}
```

Enum thực chất là các biến nguyên được gán giá trị tăng dần, mặc định từ `0`:
```c
enum WeekDay {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

int main()
{
    printf("%d %d %d %d %d %d %d",Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday); // in ra 0 1 2 3 4 5 6
    return 0;
}
```

Ta có thể gán giá trị khác thay vì giá trị mặc định từ `0`:
```c
enum WeekDay {Monday = 1, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

int main()
{
    printf("%d %d %d %d %d %d %d",Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday); // in ra 1 2 3 4 5 6 7
    return 0;
}

enum WeekDay {Monday = 1, Tuesday, Wednesday = 5, Thursday, Friday, Saturday, Sunday};

int main()
{
    printf("%d %d %d %d %d %d %d",Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday); // in ra 1 2 5 6 7 8 9
    return 0;
}
```
## 2 cách để khai báo enum

### 1. Không dùng `typedef`
```c
enum WeekDay {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday};

int main()
{
    enum WeekDay day = Monday; // nếu không dùng typedef thì khi viết phải thêm enum
    if (day == Monday)
    {
        printf("Hôm nay là thứ 2");
    }
    return 0;
}
```

### 2. Dùng từ khóa `typedef`

```c
typedef enum {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday} WeekDay;

int main()
{
    WeekDay day = Monday; // dùng typedef thì không cần thêm enum
    if (day == Monday)
    {
        printf("Hôm nay là thứ 2");
    }
    return 0;
}
```

`Lưu ý`: `typedef` không phải chỉ dùng để định nghĩa `enum`, nó là cú pháp để viết gọn, đặt tên mới cho 1 kiểu dữ liệu

Cú pháp: `typedef <kiểu dữ liệu cũ> <tên kiểu dữ liệu mới>;`

Ví dụ:
```c
typedef unsigned int ui;
typedef enum {Monday, Tuesday, Wednesday, Thursday, Friday, Saturday, Sunday} WeekDay;
typedef long long ll;
```
