# Khai báo hằng

## Dùng `#define`

Cú pháp: `#define <tên hằng> <giá trị>`

`Note:` tên hằng thường viết hoa (có thể viết thường), dùng `#define` phía sau không có `;`

```cpp
#include <iostream>
using namespace std;
#define PI 3.14
#define HANG_SO 5
```

## Dùng từ khóa `const`

Cú pháp: `const <kiểu_dữ_liệu> <tên_biến> = <giá_trị>;`

```cpp
const int HANG_SO = 5;
const double PI = 3.14;
```
